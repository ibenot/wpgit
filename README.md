##Installation via dploy.sh
1. Cloner le repo sur votre serveur web (local) du genre :
	1. 127.0.0.1/wpgit
	2. localhost:8888/wpgit
2. Ouvrir un terminal se placer dans le dossier et lancer la commande `sh dploy.sh`
3. Le script demande les identifiants de BDD et l'URL locale souhaitée et procède à l'installation.
